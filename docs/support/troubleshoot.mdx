Mengatasi jika kemungkinan terjadi masalah pada Aplikasi Dinus Metaverse sehingga dapat teratasi.

### 1. Jangan Memakai Mode Penyamaran / Incognito / Private. 
Fitur sign in dengan google tidak bisa digunakan pada mode ini.

### 2. Jangan Memakai VPN

IP dari luar indonesia terkadang di block oleh server, yang mengakibatkan asset-asset web gagal di download (aplikasi tampak blank putih)

### 3. Gunakan Browser Chrome / Edge / Mozilla dengan versi terbaru

Kami merekomendasikan untuk menggunakan browser google chrome.

### 4. Bersihkan Cache

Buka tab history pada browser lalu hapus history dan cache.
